Fathom — watch it diagnose

Fathom is a diagnostic agent that finds the root cause of a system failure. Under the hood it runs a value-of-information–guided hypothesis differential over a typed causal graph, built as a controlled evaluation.
What you're looking at & why it's interesting
That evaluation isolates what the structure buys over a bare LLM. The transferable shape is structured agentic reasoning + calibrated abstention + a controlled evaluation against real baselines; the lidar domain earns it, being hard enough — decoys, a salient red-herring trigger to demote, an abstain case — to require that structure, where an easy domain would let a bare LLM win and invalidate the result. From the reported symptom, the controller walks the typed causal graph backward from the affected KPI — traversing the structure to surface candidate subsystems — and forms competing fault hypotheses anchored to those nodes. It then repeatedly picks the single most discriminating next action by value-of-information, executes it against a deterministic evidence layer, and folds each result into a live Investigation Graph — updating belief in log-odds (evidence weighted by approximate log-likelihood ratios, aggregated to a sigmoid confidence) until one explanation dominates or it abstains rather than guess. The model supplies judgment while the tool layer supplies measurement, so a hypothesis can never rest on a reading the controller didn't actually retrieve. How to read it. The map is the controller's real reasoning substrate — the causal subgraph it walked backward from the KPI (far left) through subsystems to candidate parts (right). Lit nodes are live fault hypotheses, filled red→green by belief; dim nodes are other candidates it kept in view. Evidence attaches to the node it bears on (green = supports / red = refutes, thickness ∝ weight; pruned to the dominant association + relevant opposing side). A bright ring marks the leader; belief pulses up the headline path to the KPI. When evidence supports no single cause the harness abstains — shown honestly, no false winner.
